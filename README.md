# Bible Knowledge Graph

An interactive Streamlit app that visualises biblical characters and their relationships as a knowledge graph.

## Features
- **Knowledge Graph** tab: interactive node-link diagram (via PyVis) with colour-coded character types and relationship edges
- **Family Tree** tab: hierarchical view of biblical lineages
- Sidebar filters by character group (patriarch, prophet, apostle, etc.) and Testament (Old / New)

## Stack
- Python · Streamlit · PyVis · NetworkX

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
